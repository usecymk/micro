# Micro-Life 3D: An Autonomous Flagellated Ecosystem

Jalal Ikram, Amanda An

## 1 Project Overview

This project proposes the development of Micro-Life 3D, a real-time artificial life simulation that models autonomous flagellated microorganisms within a physically interactive three-dimensional aquatic ecosystem. Inspired by biologically grounded animation systems and behavioral ecology research, the project integrates biomechanics, hydrodynamics, perception, decision-making, and emergent group behavior into a unified simulation framework.

The simulation focuses on protozoa-like organisms such as Euglena and Monosiga, emphasizing realistic locomotion generated through deformable soft-body dynamics and flagellar propulsion. Rather than relying on predefined animation paths or scripted actions, organisms will behave autonomously through interactions between physical movement, environmental sensing, and internal motivational states.

Each microorganism will function as an independent artificial life agent capable of:

* Swimming through flexible flagellar motion  
* Detecting predators, food, and obstacles  
* Making behavioral decisions based on hunger and fear  
* Navigating dynamically in three-dimensional space  
* Exhibiting emergent behaviors such as swarming, pursuit, and evasion

The project aims to demonstrate how complex ecological behaviors can emerge naturally from relatively simple local rules and physically embodied agents.

* GPU-accelerated drawing  
* Environment rendering and interaction

The core simulation systems, including physics, locomotion, behavior, and ecosystem logic, will be implemented manually in C++ to allow greater control over optimization and artificial life behavior.

The system consists of multiple autonomous microorganism agents operating within a shared aquatic environment. Each organism contains:

* A deformable mass-spring body  
* A flagellar locomotion controller  
* Sensory perception modules  
* Behavioral decision-making logic

At each simulation step, organisms:

2. Update internal states such as hunger and fear

## 4 System Design

### Biomechanical Soft-Body Simulation

Each microorganism will be represented using a deformable mass-spring model composed of interconnected nodes and springs.

The body structure includes:

* Structural springs for maintaining body shape  
* Damping springs for numerical stability  
* Muscle-actuator springs for locomotion

The flagellum generates propulsion through oscillatory sinusoidal motion. Traveling wave motion along the tail produces thrust against the surrounding fluid, allowing organisms to swim realistically through the environment.

### Hydrodynamic Approximation

Instead of computationally expensive fluid simulation, the system approximates hydrodynamic interaction using surface-based drag and thrust forces applied to triangular body elements.

This provides convincing fluid-like behavior while remaining computationally efficient enough for real-time simulation.

### Sensory Perception

Each organism includes:

* Spherical visual sensing  
* Obstacle detection  
* Predator and prey detection  
* Nutrient and temperature sensing

The perceptual system uses selective attention filtering to prioritize stimuli relevant to the organism’s current state. For example:

* Hungry organisms prioritize food detection  
* Fearful organisms prioritize predators  
* Wandering organisms ignore distant irrelevant objects

### Behavioral Intelligence

Behavior selection is governed by a hierarchical intention system. Organisms evaluate sensory information and internal motivational variables before selecting actions.

Primary behavioral priorities include:

1. Avoid collisions  
2. Escape predators  
3. Seek food  
4. Wander

Agents also maintain a lightweight short-term memory system allowing interrupted goals to resume after danger or navigation adjustments.

### Ecosystem Dynamics

The simulation includes multiple interacting species:

* Small swarming protozoa  
* Larger predatory microorganisms

Predators evaluate prey based on energetic efficiency and movement cost, while prey organisms coordinate movement using local swarming rules including:

* Alignment  
* Cohesion  
* Separation  
* Velocity matching

These local interaction rules are expected to produce emergent ecological behaviors such as:

* Dynamic swarm formation  
* Predator-prey pursuit  
* Group evasion  
* Collective navigation around obstacles
